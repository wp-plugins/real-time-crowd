The official RealTimeCrowd.net plugin for Wordpress.

== Description ==

This plugin enables Real-Time Crowd tracking and generation on your WordPress site.

== Installation ==

1. Upload the 'real-time-crowd' directory into the '/wp-content/plugins/' directory on your server.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Using the "WordPress *General* Settings" menu, navigate to Real-Time Crowd and enter your RealTimeCrowd.net Account alias.
